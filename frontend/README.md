### PEMESANAN TIKET BUS - FRONTEND

# Deskripsi Project
Project ini dibuat berdasarkan permasalahan yang saya alami, yaitu proses pemesanan tiket bus yang masih dilakukan secara offline dan kurangnya media informasi yang terstruktur.

Melalui sistem ini, saya mengembangkan aplikasi berbasis web untuk:
- Mendigitalkan jadwal dan transaksi tiket secara real-time
- Memberikan kepastian tiket bagi penumpang
- Meningkatkan efisiensi kerja agen tanpa proses manual

Project ini juga bertujuan untuk mengasah kemampuan saya dalam pengembangan frontend serta integrasi dengan backend API.

# Fitur 
- Login
- Register
- Profile
- Pemesanan Tiket
- Reset Pass
- Integrasi API

# Tech
